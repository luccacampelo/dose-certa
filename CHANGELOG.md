# Changelog

Registro das mudancas relevantes do projeto.

## [1.0.0] - 2026-03-25
### Added
- Estrutura inicial do projeto DoseCerta
- Interface GUI web com Streamlit
- Cadastro e listagem de medicamentos
- Registro de doses tomadas e status diario (pendente, atrasada, tomada)
- Persistencia local em arquivo JSON
- Testes automatizados com Pytest
- Linting com Ruff
- CI com GitHub Actions (push e pull_request)

### Documentation
- README com instrucoes de instalacao, execucao, testes e lint
- Badge de build do GitHub Actions no README
