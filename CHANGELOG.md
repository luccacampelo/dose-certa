# Changelog

Todas as mudanças relevantes deste projeto serão registradas neste arquivo.

O formato segue o princípio de versionamento semântico.

## [1.0.0] - 2026-03-25
### Added
- Estrutura inicial do projeto `DoseCerta`
- Interface GUI web com Streamlit
- Cadastro e listagem de medicamentos
- Registro de doses tomadas e cálculo de status diário
- Persistência em arquivo JSON local
- Testes automatizados com Pytest
- Linting com Ruff
- Pipeline de CI com GitHub Actions
